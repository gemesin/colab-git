Github

GitHub adalah layanan penginangan web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem pengontrol versi Git dan layanan hosting internet

Didirikan : 2008

Kantor Pusat : San Francisco, California, Amerika

Pendiri : Tom Preston-Werner, Chris Wanstrath, Scott Chacon, P. J. Hyett

Anak perusahaan : Easel Inc.

Organisasi induk: Microsoft
